## Installation

1. Clone the repo

```sh
git clone https://github.com/pierrelstan/echo.git
```

2. Open the project Directory in your local machine

```
cd echo
```

1. install packages

```sh
yarn install
```

1. Run Application

```sh
yarn run dev
```
